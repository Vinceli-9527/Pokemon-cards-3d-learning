# pokemon-cards-3d-learning

> 🎓 自制学习项目 · 宝可梦卡牌 3D 全息特效展示

一个**借鉴学习**自 [simeydotme/pokemon-cards-css](https://github.com/simeydotme/pokemon-cards-css) 的自制项目。鼠标移到卡片上会看到立体倾斜和光泽流动，点击卡片可以放大查看细节。

## 如何运行

### 你需要准备

- 一台 Windows 电脑
- 本项目文件夹（已经在你电脑上了）

### 三步启动

1. 打开本项目文件夹
2. 双击 `start.bat`
3. 浏览器会自动打开 `http://localhost:3000`

> 如果没有自动打开浏览器，请手动打开浏览器，在地址栏输入 `http://localhost:3000` 并回车。

### 如何停止

在启动时弹出的命令行窗口中按 `Ctrl + C`，然后关闭窗口即可。

## 页面操作

| 操作 | 效果 |
|---|---|
| 鼠标在卡片上移动 | 卡片跟随倾斜，全息光泽流动 |
| 点击卡片 | 卡片放大，立体效果增强 |
| 再次点击 / 点击空白处 / 按 Esc | 卡片缩回原状 |
| 点击 ← → 箭头按钮 | 切换到上一张 / 下一张卡片 |
| 按键盘 ← → 键 | 同上 |
| 手机触屏滑动 | 卡片跟随倾斜 |

## 常见问题

**Q: 页面打开了但卡片没有 3D 效果？**

刷新页面（按 `Ctrl + Shift + R` 强制刷新），清除浏览器缓存。

**Q: 页面显示"加载卡片数据失败"？**

确认 `data` 文件夹和 `media` 文件夹与 `index.html` 在同一目录下，没有被移动或删除。

**Q: 可以在手机上查看吗？**

可以。确保手机和电脑在同一 Wi-Fi 网络下，用手机浏览器访问 `http://你的电脑IP:3000`（在电脑命令行中输入 `ipconfig` 查看 IP 地址）。页面会自动适配手机屏幕。

**Q: 启动时提示端口被占用？**

`start.bat` 会自动关闭占用端口的程序。如果仍有问题，重启电脑后再试。
